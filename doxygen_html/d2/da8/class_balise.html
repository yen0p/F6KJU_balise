Description détaillée

Une "fiche de réglages" pour une émission.

Cette classe n'est pas un "travailleur" mais plutôt un conteneur, une sorte de fiche cartonnée qui regroupe toutes les informations nécessaires pour une émission sur une bande spécifique : le message, la fréquence, et les broches à activer.
