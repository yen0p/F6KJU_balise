Description détaillée

Le pilote du composant qui génère le signal radio (DDS AD9850).

Cette classe est une "télécommande" pour le module AD9850. Elle simplifie son utilisation en cachant les détails techniques complexes. On peut simplement lui demander de générer une fréquence, et elle se charge d'envoyer les bons ordres au composant.

Valide la nouvelle fréquence.

Après avoir envoyé le code, une petite impulsion sur cette broche dit au DDS : "Ok, c'est bon, tu peux utiliser cette nouvelle fréquence maintenant".
