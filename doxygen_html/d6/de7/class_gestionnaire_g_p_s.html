Description détaillée

Spécialiste de la communication avec le module GPS.

Cette classe est un "traducteur". Elle reçoit les phrases complexes (trames NMEA) envoyées par le module GPS et les déchiffre pour en extraire des informations simples et utilisables : l'heure, la date, la position, etc.
